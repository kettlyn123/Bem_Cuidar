## 🔧 TAREFAS COMUNS

### Quero mudar a cor do app
👉 Abra `css/styles.css` linha 10-11

### Quero adicionar um novo campo no formulário
👉 Leia [GUIA_EDICAO.md](GUIA_EDICAO.md) seção "Adicionar um novo campo"

### Meus dados desapareceram
👉 Abra o console (F12) e execute:
```javascript
localStorage.removeItem('bem_cuidar_medicines');
location.reload();
```

### Quero criar uma nova página
👉 Leia [GUIA_EDICAO.md](GUIA_EDICAO.md) seção "Adicionar uma página nova"

### O app está muito lento
👉 Limpe o cache: Ctrl+Shift+Delete
